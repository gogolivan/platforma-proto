=== Buf
==== Help
[source,shell]
----
docker-compose run --rm --remove-orphans buf --help
----

==== Init
[source,shell]
----
docker-compose run --rm --remove-orphans buf init
----

==== Format
[source,shell]
----
docker-compose run --rm --remove-orphans buf format
----

==== Lint
[source,shell]
----
docker-compose run --rm --remove-orphans buf lint
----
